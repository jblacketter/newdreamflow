# Background Music Files

This directory contains ambient music files for the dream journal app.

## Music Options:
- ambient_forest: Soothing forest sounds
- meditation_bells: Soft meditation bells
- ocean_waves: Calming ocean waves
- rain_sounds: Gentle rain
- tibetan_bowls: Tibetan singing bowls
- white_noise: Soft white noise
- none: No music

Note: The actual audio files need to be added to this directory with the following naming convention:
- ambient_forest.mp3
- meditation_bells.mp3
- ocean_waves.mp3
- rain_sounds.mp3
- tibetan_bowls.mp3
- white_noise.mp3

These files should be royalty-free ambient sounds suitable for relaxation and dream journaling.
